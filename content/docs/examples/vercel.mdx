在 Vercel 部署任何项目后，您可以使用自定义域名。

首先访问 https://vercel.com/[account]/[project]/settings/domains。

添加您的自定义域名，如 `beta.kedaya.xyz`：

![](/_static/examples/vercel_01.png)

然后在 kedaya.xyz 上配置 CNAME 内容：

![](/_static/examples/vercel_02.png)

等待 DNS 记录在 Vercel 上生效：

![](/_static/examples/vercel_03.png)

完成！

### 域名验证

当您首次申请自定义域名时，需要在 `dashboard/records` 添加一条 `TXT` 记录以验证域名所有权。

`TXT` 记录的 `name` 是 `_vercel`，`value` 是 `vs-domain-verify=xxxxxx`，您可以在 `https://vercel.com/[account]/[project]/settings/domains` 找到它们。
