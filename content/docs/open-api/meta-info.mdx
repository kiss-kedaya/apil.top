##  使用方法

API URL:
```bash
https://qali.cn/api/v1/scraping/meta?url=https://www.example.com&key=your_api_key
```

API 响应:
```json
{
  title: "",
  description: "",
  image: "",
  icon: "",
  url: "",
  lang: "",
  author: "",
  timestamp: "",
  payload: "",
}
```

示例:

[https://qali.cn/api/v1/scraping/meta?url=https://vmail.dev&key=example_key](https://qali.cn/api/v1/scraping/meta?url=https://vmail.dev&key=b6e1c218-c090-434d-ba3f-97fb48fbccd9)
