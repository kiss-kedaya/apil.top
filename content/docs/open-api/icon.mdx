## 使用方法

```bash
https://qali.cn/api/v1/icon
```

### 查询参数

- `type` (可选): 图标类型。默认为 `text`。
- `w` (可选): 图标宽度。默认为 `128`。
- `h` (可选): 图标高度。默认为 `64`。
- `value`: 图标值。
- `animate` (可选): 是否为图标添加动画。默认为 `false`。
- `fillType` (可选): 图标填充类型。默认为 `Linear`。
- `primaryColor` (可选): 图标主要颜色。默认为 `#FC466B`。
- `secondaryColor` (可选): 图标次要颜色。默认为 `#3F5EFB`。
- `angle` (可选): 图标角度。默认为 `45`。
- `radialGlare` (可选): 是否应用径向眩光。默认为 `false`。
- `radius` (可选): 图标圆角半径。默认为 `12`。
- `strokeSize` (可选): 描边大小。默认为 `0`。
- `strokeColor` (可选): 描边颜色。默认为 `#FFFFFF`。
- `strokeOpacity` (可选): 描边不透明度。默认为 `100`。
- `color` (可选): 图标颜色。默认为 `#FFFFFF`。
- `size` (可选): 图标大小。默认为 `15`。
- `family` (可选): 图标字体系列。默认为 `sans-serif`。
