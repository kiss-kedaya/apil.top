## 使用方法

API URL:
```bash
https://kedaya.xyz/api/v1/scraping/screenshot?url=https://vmail.dev&key=your_api_key
```

### 查询参数
- `url` (必需): 要捕获的有效 URL。
- `width` (可选): 输出截图的宽度。默认为 `250`。
- `height` (可选): 输出截图的高度。默认为 `250`。
- `viewportWidth` (可选): 渲染视口的宽度。默认为 `1080`。
- `viewportHeight` (可选): 渲染视口的高度。默认为 `1080`。
- `forceReload` (可选): 强制重新加载缓存的图像。默认为 `false`。
- `isMobile` (可选): 向用户代理添加移动标志。默认为 `false`。
- `isFullPage` (可选): 渲染整个页面而不是视口裁剪。默认为 `false`。
- `isDarkMode` (可选): 偏好深色配色方案。默认为 `false`。
- `deviceScaleFactor` (可选): 指定设备缩放因子（可视为 DPR）。默认为 `1`。

```bash

https://kedaya.xyz/api/v1/scraping/screenshot?url=https://vmail.dev&key=example_key&width=500&height=500
https://kedaya.xyz/api/v1/scraping/screenshot?url=https://vmail.dev&key=example_key&viewportWidth=1920&viewportHeight=1080&isFullPage=true
https://kedaya.xyz/api/v1/scraping/screenshot?url=https://vmail.dev&key=example_key&viewportWidth=1920&viewportHeight=1080&isFullPage=true&isDarkMode=true
```
