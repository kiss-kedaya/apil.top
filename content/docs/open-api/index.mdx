## 什么是开放 API？

抓取 API 允许您快速提取有价值的结构化网站数据。它是一个强大的工具，可用于从网站提取信息并将其转换为结构化格式。

apil.top 提供两种简单易用的网站数据抓取 API：

- [截图 API](/docs/open-api/screenshot)
- [元数据 API](/docs/open-api/meta-info)
- [URL 转 Markdown API](/docs/open-api/markdown)
- [URL 转文本 API](/docs/open-api/text)

## 如何使用开放 API？

按照 [Dashboard/Scraping-API](/dashboard/scrape) 或 [截图](/docs/open-api/screenshot) 开始使用。

## API 密钥

所有用途都需要 API 密钥。您可以从 [Dashboard/Settings](/dashboard/settings) 生成您的 API 密钥。

## API 速率限制

API 没有速率限制。
