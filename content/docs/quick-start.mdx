欢迎使用 apil.top，这是一个基于 Cloudflare DNS API 的多租户 DNS 记录管理系统，同时提供 URL 短链接功能。本指南将帮助您快速开始使用我们的网站。

## 注册和登录

1. **访问网站**：首先，访问 https://apil.top 。
3. **注册/登录您的账户**：使用注册的电子邮件或 Google/Github 账户在[登录页面](https://apil.top/login)登录。

## 生成短链接

1. **导航到短链接页面**：在导航栏中选择[短链接](https://apil.top/dashboard/urls)选项。
2. **输入长 URL**：将您想要缩短的 URL 粘贴或输入到提供的输入框中。
3. **生成短链接**：点击"生成"按钮，系统将自动为您创建一个短链接。
4. **复制并使用**：点击"复制"按钮将短链接复制到剪贴板，然后可以在需要的地方使用它。

## 创建电子邮件地址

1. **导航到电子邮件管理页面**：登录后，您将在导航栏中看到"电子邮件管理"选项，点击[电子邮件管理](https://apil.top/emails)继续。
2. **添加电子邮件**：点击"创建新电子邮件"按钮创建一个新的电子邮件地址。
3. **准备使用**：创建电子邮件地址后，您可以开始使用它进行电子邮件通信。

## 创建聊天室

1. **导航到聊天室页面**：登录后，您将在导航栏中看到"聊天室"选项，点击[聊天室](https://apil.top/chat)继续。
2. **分享房间**：点击"分享房间"按钮复制房间链接并与他人分享。
3. **连接房间**：一旦其他用户加入房间，他们就可以开始聊天。

## 创建 DNS 记录

1. **导航到 DNS 管理页面**：登录后，您将在导航栏中看到"DNS 管理"选项，点击[DNS 管理](https://apil.top/dashboard/records)继续。
2. **添加记录**：点击"添加记录"按钮并选择您需要的记录类型（如 CNAME、A 等）。
3. **填写详细信息**：按照提示输入主机名、目标地址和其他详细信息。
4. **保存记录**：确认所有内容无误后，点击"保存"按钮。

## 实时更新

- **DNS 记录变更**：一旦您更新或添加新的 DNS 记录，系统将立即开始传播这些变更。
- **短链接访问**：生成的短链接可以立即使用，无需额外等待时间。
