## 功能特点

- 🔗 **URL 短链接：** 生成带访问统计和密码的短链接（支持API）
- 📮 **邮件支持：** 接收和发送邮件（支持API）
- 💬 **P2P 聊天：** 即时开始聊天
- 🌐 **多租户支持：** 无缝管理多个DNS记录
- 📸 **截图 API：** 访问截图API、网站元信息抓取API
- 😀 **权限管理：** 便捷的管理员面板用于审核
- 🔒 **安全可靠：** 基于Cloudflare强大的DNS API构建

## 自托管教程

### 要求

- [Vercel](https://vercel.com) 用于部署应用
- [Cloudflare](https://dash.cloudflare.com/) 账户  
- 一个托管在Cloudflare上的**域名**

查看[开发者文档](/docs/developer/installation)了解更多。

## 本地开发

复制 `.env.example` 到 `.env` 并填写必要的环境变量。

```bash
git clone https://github.com/kiss-kedaya/kedaya.xyz
cd kedaya.xyz
pnpm install

# 在 localhost:3000 上运行
pnpm dev
```
