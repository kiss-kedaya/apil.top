---
title: Cloudflare 配置
description: 如何配置 Cloudflare API。
---

开始之前，您必须拥有 Cloudflare 账户并托管在 Cloudflare 上。

在本节中，您可以更新这些变量：

```js title=".env"
CLOUDFLARE_ZONE_ID=abcdef1234567890
CLOUDFLARE_API_KEY=1234567890abcdef1234567890abcdef
CLOUDFLARE_EMAIL=user@example.com
CLOUDFLARE_ZONE_NAME=qali.cn
```

### 变量说明

- **CLOUDFLARE_ZONE_ID**：这是您 Cloudflare 区域的唯一标识符。您可以在 Cloudflare 仪表板的域名概览部分找到它。

> 按照 [这种方式](https://dash.cloudflare.com/Your_Acount_Id/qali.cn)，向下滚动至 `Zone ID`。
  
- **CLOUDFLARE_API_KEY**：这是您用于验证 Cloudflare API 请求的 API 密钥。您可以在 Cloudflare 仪表板的 `profile` -> `api-tokens` 部分生成或找到您的 API 密钥。

> 访问 [https://dash.cloudflare.com/profile/api-tokens](https://dash.cloudflare.com/profile/api-tokens)，向下滚动至 `API Token`，应使用 `Global API Key`。
  
- **CLOUDFLARE_EMAIL**：这是与您的 Cloudflare 账户关联的电子邮件地址。它与 API 密钥一起用于认证。

- **CLOUDFLARE_ZONE_NAME**：这是您 Cloudflare 区域的名称。它用于在 Cloudflare API 请求中指定区域。
