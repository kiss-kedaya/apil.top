## 步骤

<Callout type="note">
  电子邮件部分与 [resend](https://resend.com/) 文档类似。
  如果需要，您可以在 [此处](https://authjs.dev/getting-started/installation#setup-environment) 查找官方文档。
</Callout>

<Steps>

### 创建账户

如果您在 Resend 上没有账户，请在注册后按照 [此处](https://resend.com/signup) 的步骤操作。

### 创建 API 密钥

登录 Resend 后，系统会建议您创建第一个 API 密钥。

复制/粘贴到您的 `.env` 文件中。

```js
RESEND_API_KEY = re_your_resend_api_key;
```

</Steps>
