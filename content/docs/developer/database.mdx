title: 数据库
description: 如何配置 Neon 数据库。
---

以下是一些免费的 postgres 数据库：
- [Neon](https://neon.tech)
- [Supabase](https://supabase.com)

## 步骤

<Steps>

### 创建 Neon 账户

如果您没有 Neon 账户，请按照[这里](https://neon.tech/docs/get-started-with-neon/signing-up)的步骤创建一个。

### 复制数据库 URL

在 Neon 上创建项目后，您可以复制数据库 URL。从列表中选择 `Prisma`，然后选择 `env` 标签。

<Callout type="warning">
  <b>不要忘记</b>使用眼睛图标显示隐藏在星号后面的信息。在暗模式下不太容易看见。
</Callout>

粘贴到您的 `.env` 文件中。

```js
DATABASE_URL = "postgres://alex:AbC123dEf@ep-cool-darkness-123456.us-east-2.aws.neon.tech/dbname";
```

### 推送迁移

您可以使用命令行推送迁移到数据库。

```bash
npx prisma db push
```

</Steps>
