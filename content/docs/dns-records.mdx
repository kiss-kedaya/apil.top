## 什么是 DNS 记录？

DNS 记录是存储在 DNS 区域中的一条信息。它是包含主机名和 IP 地址的记录。DNS 记录可以是 CNAME 或 A 记录。

## 何时使用 DNS 记录？

- 当您需要将流量从一个域名重定向到另一个域名时。
- 当您需要将流量从一个 IP 地址重定向到另一个 IP 地址时。
